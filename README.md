# 🏡 Real Estate Portfolio Dashboard - Power BI

## 📌 Project Overview

The **Real Estate Portfolio Dashboard** is an interactive Power BI project designed to analyze property sales, availability, revenue, demand scores, and market trends. It provides valuable insights into property performance across cities and property types, enabling data-driven decision-making for real estate businesses.


## 🚀 Features

* Total Properties KPI
* Total Sales KPI
* Available Properties KPI
* Revenue KPI
* City-wise Average Property Price
* Property Count by Sales Status
* Property Count by Property Type
* Demand Score vs Price Analysis
* Property Count by Year Built
* Interactive Filters (City, Property Type, Sales Status, Bedrooms)
